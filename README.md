Web-licor

La aplicación web está diseñada como una plataforma de comercio electrónico especializada en la venta de bebidas alcohólicas. El sistema implementa un modelo de gestión basado en roles, diferenciando las funcionalidades disponibles para clientes y administradores.

El rol de cliente permite consultar el catálogo de productos disponibles, visualizar la información de cada artículo y realizar el proceso de compra a través de la plataforma.

Por su parte, el panel de administración cuenta con un sistema de autenticación mediante usuario y contraseña, garantizando el acceso exclusivo a usuarios autorizados. Desde este módulo es posible gestionar el inventario mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre los productos, permitiendo agregar nuevos artículos, modificar la información existente y eliminar registros cuando sea necesario.

Adicionalmente, la aplicación incorpora un módulo de inspección de datos que permite visualizar las tablas de la base de datos correspondientes a productos y usuarios, facilitando la consulta y verificación de la información almacenada en el sistema.
